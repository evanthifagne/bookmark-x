# HeroUI v3 coche toutes les cases du design system moderne : OKLCH, headless, parité Figma 1:1 et outillage agent (MCP, skills, llms.txt)

**Résumé** — HeroUI v3 : réécriture complète pour React (75+ composants) plus une lib React Native neuve (37+), Tailwind v4 avec variables CSS et couleurs OKLCH, animations CSS sans runtime JS, accessibilité via React Aria, architecture compound/headless. Deux signaux d'époque : la parité 1:1 entre le kit Figma et le code, et une couche « IA-native » assumée — serveur MCP, Agent Skills, llms.txt — pour que les agents consomment le système directement.
