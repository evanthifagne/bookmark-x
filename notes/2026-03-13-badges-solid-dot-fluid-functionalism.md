# Un composant Badge réduit à l'essentiel : deux variantes (solid, dot), trois tailles, toutes les couleurs Tailwind

**Résumé** — Le Badge du système « Fluid Functionalism » de Micka : variante solid (fond teinté) ou dot (pastille colorée + libellé neutre), trois tailles (sm/md/lg), 17 couleurs Tailwind. API minimale (variant, size, color), installable via le registre shadcn (`npx shadcn add …/r/badge.json`) et « primitive-agnostic ». Bon exemple d'un composant de statut spécifié proprement sans sur-ingénierie.
