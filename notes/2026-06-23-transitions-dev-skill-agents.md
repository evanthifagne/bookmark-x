# transitions.dev : 21 transitions UI essentielles, copiables en CSS/React ou installables comme skill d'agent

**Résumé** — Jakub Antalik enrichit transitions.dev : 21 transitions UI prêtes à l'emploi (resize de carte, flip de chiffres avec blur, badge de notif à ressort, modal, confettis physiques, shake d'erreur…), copiables en CSS ou React. Le point notable : la collection s'installe aussi comme skill pour Claude, Cursor ou Codex (`npx skills add jakubantalik/transitions`) — la référence motion devient consommable par les agents.
