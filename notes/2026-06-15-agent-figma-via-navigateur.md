# Donner un navigateur à son agent + le global window.figma : Figma devient entièrement automatisable, au-delà des limites du MCP

**Résumé** — Astuce : donner à un coding agent l'accès à un navigateur et lui dire d'utiliser le global `window.figma` (miroir de l'API plugin) — il peut alors automatiser à peu près tout dans Figma. Le tweet cité raconte le déclencheur : Codex a atteint la limite du MCP Figma et a simplement ouvert Figma dans un onglet pour continuer. Une équipe design entière aurait acheté Do Browser juste pour ça.
