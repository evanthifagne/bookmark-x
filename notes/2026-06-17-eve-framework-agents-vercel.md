# eve, « le Next.js des agents » : un framework à conventions de fichiers où la production est déjà câblée

**Résumé** — Vercel lance eve, framework open source pour construire, exécuter et scaler des agents. Le parti pris : définir ce que fait l'agent via une structure de dossiers conventionnelle (agent.ts, instructions.md, tools/, skills/, sandbox/, schedules/) sans assembler soi-même l'infra — exécution durable et production intégrées d'office, comme Next.js l'a fait pour le web.
