# hue : une skill open source qui transforme une marque (URL ou screenshot) en design system complet pour Claude Code

**Résumé** — hue (hueapp.io, open source, gratuit) s'installe comme skill de Claude Code ou Codex : on lui donne un lien de marque, un nom ou une capture, et il génère en un coup un design system complet — tokens, typographie, radii, élévations, ~40 composants avec variantes, modes clair et sombre, dont une bibliothèque de composants navigable en HTML. Ensuite, chaque UI que l'agent produit respecte cette marque. C'est la réponse directe au problème du « generic AI UI ».
