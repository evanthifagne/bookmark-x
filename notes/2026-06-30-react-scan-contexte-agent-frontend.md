# Donner react-scan et un harnais navigateur à un agent frontend améliore drastiquement la qualité de son contexte

**Résumé** — Un agent de code frontend travaille à l'aveugle s'il ne voit que les fichiers. En lui donnant react-scan (qui visualise les re-renders et problèmes de performance React en direct) plus un harnais navigateur / computer use, il observe le comportement réel de l'UI et son contexte devient bien plus riche que le seul code source.

**Pourquoi gardé** — [hypothèse : s'inscrit dans ton thème « outils-ia » — équiper tes agents de boucles de feedback visuelles concrètes plutôt que de les laisser deviner depuis le code.]
