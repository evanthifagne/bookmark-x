# La zone de pouce couvre mal le tiers supérieur de l'écran : y placer la lecture, pas l'action

**Résumé** — Sur un téléphone tenu à une main, ~49 % de l'écran est confortablement atteignable au pouce. Les actions primaires doivent vivre dans le tiers inférieur ; le tiers supérieur est pour le contenu à lire, pas à toucher. Taille de cible minimale recommandée : 44×44 pt (Apple) / 48×48 dp (Material).

**Pourquoi gardé** — Checklist rapide pour les revues de maquettes mobiles Prox.ia.
