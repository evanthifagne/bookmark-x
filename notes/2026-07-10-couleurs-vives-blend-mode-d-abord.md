# Pour des couleurs vives en UI : appliquer un blend mode d'abord, extraire les couleurs plates ensuite

**Résumé** — Technique de Sebastiano Guerriero (CodyHouse) : plutôt que de choisir ses couleurs à la main, il applique un mode de fusion (blend mode) pour générer des teintes riches, puis en extrait les valeurs plates à utiliser telles quelles dans l'UI. Le blend mode sert d'outil de génération, pas de rendu final.
