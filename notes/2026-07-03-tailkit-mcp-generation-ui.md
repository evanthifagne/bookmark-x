# Le MCP de Tailkit ancre la génération d'UI dans une collection de composants faits main, au lieu de laisser l'IA improviser

**Résumé** — Tailkit propose un serveur MCP qui fait générer l'UI d'un SaaS à partir de sa grande collection de composants « handcrafted ». L'idée de fond : la réponse au « UI slop » n'est pas un meilleur prompt mais un meilleur socle — l'agent assemble des composants éprouvés plutôt que d'inventer du markup. Même pattern que HeroUI Native ou Originkit : le design system distribué à l'agent via MCP.
