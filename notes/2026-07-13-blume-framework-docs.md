# Blume : le framework de docs qui coche tout — beau, rapide, Astro, MIT — face à Starlight, Mintlify et Fumadocs

**Résumé** — Blume (useblume.dev) génère un site de documentation complet depuis un simple dossier de Markdown : navigation, recherche, images OG et llms.txt inclus, zéro boilerplate d'app. Le comparatif du relayeur situe le créneau : Starlight rapide mais daté, Mintlify beau mais cher, Fumadocs beau mais lent — Blume serait beau, rapide, Astro et MIT. Anti lock-in : `blume eject` reconvertit le tout en projet Astro nu.
