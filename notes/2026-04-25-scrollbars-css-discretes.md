# Deux lignes de CSS suffisent pour des scrollbars discrètes : scrollbar-width + scrollbar-color

**Résumé** — Pour que les barres de défilement « ne gâchent pas » une interface : `scrollbar-width: thin;` et `scrollbar-color: gray transparent;`. Déployé sur Sentry, avec un effet visuel immédiat sur toute l'app. Un micro-détail d'UI au ratio effort/impact excellent.
