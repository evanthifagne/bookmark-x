# gradient-shimmer : un shimmer de texte multi-stop en package React, pensé pour les états « thinking » des UI d'IA

**Résumé** — Package npm de shimmer de texte à dégradé multi-stop pour React (gradient-shimmer.vercel.app), avec preview dark et courbe d'easing réglable. Le tweet cité montre l'usage d'origine : appliquer ces dégradés au canal « thinking » d'une UI d'agent pour le distinguer visuellement des messages normaux — le shimmer devient un signal d'état, pas une décoration.
