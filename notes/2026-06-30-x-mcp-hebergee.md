# X publie son MCP hébergé : l'API X (posts, bookmarks, tendances) accessible à n'importe quel agent sans setup

**Résumé** — X annonce un MCP officiel hébergé : Grok, Cursor ou tout outil compatible MCP peut appeler l'API X sans configuration — recherche de posts, profils, bookmarks, tendances, news, Articles (serveur api.x.com/mcp, plus un Docs MCP pour la documentation). L'argument : donner aux agents « la meilleure source d'information temps réel au monde ». Directement pertinent pour un pipeline qui consomme des bookmarks X.
