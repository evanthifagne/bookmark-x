# Taste Skill : encoder du « goût » dans un fichier SKILL.md pour empêcher les agents de générer du frontend générique

**Résumé** — Taste Skill (tasteskill.dev) est un ensemble de skill files open source (« The Anti-Slop Frontend Framework for AI Agents ») qui s'installe en une commande (`npx skills add Leonxlnx/taste-skill`) dans Cursor, Claude Code, Codex, v0, Lovable… La v2 lit le brief, en infère une direction de design, applique un vrai design system quand c'est pertinent, fait un audit d'abord sur les redesigns et un pre-flight check strict. Le pari : le goût est transmissible comme instruction versionnée.

**Pourquoi gardé** — [hypothèse : s'inscrit dans ton thème « outils-ia » — la lutte contre le « AI slop » visuel via skills, directement pertinent pour tes workflows de génération d'UI.]
