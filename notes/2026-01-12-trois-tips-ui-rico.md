# Trois micro-règles d'UI : paire de fontes précise, icônes 16px/1.2px, fond gris derrière les interfaces denses en données

**Résumé** — Trois réglages concrets et directement réutilisables : ES Rebond Grotesque pour les titres avec ABC Diatype pour le texte courant ; des icônes calibrées à 16 px avec un trait de 1,2 px ; et systématiquement un fond gris derrière une interface chargée en données, pour que les surfaces blanches (cartes, tableaux) se détachent.
