# L'IA sait générer du code d'animation mais pas « sentir » le timing : une skill lui enseigne les 12 principes Disney

**Résumé** — Constat : les skeleton loaders générés par IA sont presque toujours ratés, car le modèle produit du code d'animation sans intention de mouvement. La réponse : une skill open source (LottieFiles/motion-design-skill, ~650 étoiles) qui enseigne aux agents les 12 principes d'animation de Disney adaptés à l'UI — timing, easing, chorégraphie. Compatible CSS, GSAP, Framer Motion et Lottie ; le même prompt donné à Claude Code avec et sans la skill produit des résultats très différents.

**Pourquoi gardé** — [hypothèse : s'inscrit dans ton thème « outils-ia » — équiper tes agents de connaissances design (motion) plutôt que d'espérer qu'ils les devinent.]
