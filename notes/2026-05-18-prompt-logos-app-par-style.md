# Générer un logo d'app « premium » en 10 secondes : donner des icônes de référence et demander le même style exact

**Résumé** — La technique tient dans le prompt : fournir au modèle une planche d'icônes d'apps au style recherché et demander « generate a high-quality app icon for 'X app', following the exact same visual style ». Le transfert de style par l'exemple remplace la description verbeuse du rendu voulu.

**Pourquoi gardé** — [hypothèse : s'inscrit dans ton thème « outils-ia » — prompter par référence visuelle plutôt que par adjectifs, un réflexe transposable à toute génération design.]

**Étapes clés**

1. Rassembler des icônes d'apps du style visé (planche de référence).
2. Prompt : « Look at this app icon styles and generate a high-quality app icon for 'X app', following the exact same visual style. »
