# Transitions.dev publie une skill qui apprend aux agents les transitions produit

**Résumé** — Jakub Antalik condense sa bibliothèque de transitions produit (transitions.dev) en une skill installable : `npx skills add jakubantalik/transitions.dev`. Elle enseigne à l'IA le motion produit et embarque toutes les transitions du site ; compatible Claude Code, Cursor, GitHub Copilot, Codex et Gemini CLI.
