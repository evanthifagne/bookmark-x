# Servir du Markdown aux agents IA via les en-têtes Accept : un standard émergent, et un validateur

**Résumé** — Un site web peut négocier son contenu : renvoyer du Markdown (léger, structuré) quand c'est un agent IA qui demande, du HTML pour les humains, via l'en-tête HTTP Accept. acceptmarkdown.com (Ben Word) vérifie qu'un site l'implémente correctement — Vary, q-values, 406, `Link rel=alternate` — et suit quels agents IA adoptent réellement le standard, avec des guides d'intégration. Le nouveau site de Cloudflare sert de référence.
