# Un setup d'orchestrateur d'agents maison : Claude Code sur serveur, une UI par business unit, pilotage depuis Telegram

**Résumé** — Léo Bachelot décrit sa plateforme d'orchestration d'agents auto-hébergée, en forkant Paperclip : Claude Code hébergé sur Hetzner attaqué en SQL, noVNC pour contrôler un navigateur à distance, connecteurs MCP, bases Supabase compartimentées par business unit, Telegram/PWA pour parler aux agents depuis le téléphone, dashboard sur Netlify/Tailscale. L'argument de fond : ce genre de setup reste niche mais creuse massivement l'écart de productivité.
