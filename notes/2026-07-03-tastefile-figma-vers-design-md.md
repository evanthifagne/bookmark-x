# Tastefile compile le CSS d'une frame Figma en un design.md que l'agent de code suit — le « goût » transmis en fichier

**Résumé** — Tastefile (usetastefile.com) prend le CSS copié d'une frame Figma riche et le compile en un design.md structuré (système de couleurs, radius, typo…) que Claude ou Cursor suivent pour rester on-brand. L'idée forte : le goût d'un design qu'on aime devient un artefact textuel réutilisable, le chaînon manquant entre une référence visuelle et un agent de code.

**Pourquoi gardé** — [hypothèse : s'inscrit dans ton thème « design-systems » — formaliser une direction visuelle en fichier lisible par les agents, exactement le pont Figma ↔ code que tu explores.]
